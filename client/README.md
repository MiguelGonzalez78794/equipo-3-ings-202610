# UPB Primíparos

App web para estudiantes nuevos de la Universidad Pontificia Bolivariana - Medellín.

## Funcionalidades
- Login y registro con Supabase Auth
- Mapa del campus
- Directorio de profesores
- Preguntas frecuentes

## Stack
- React + Vite
- react-router-dom
- Supabase (Auth + PostgreSQL)

## Para correr el proyecto

1. Entra a la carpeta front
cd front
npm install
2. Crea un archivo .env en front/ con las claves de Supabase (se las pase por teams)
3. npm run dev
