# Design Documentation - AGE System

## Overview

Esta sección contiene toda la documentación relacionada con la fase de diseño del sistema AGE. Aquí se describen las decisiones de experiencia de usuario, interfaz, arquitectura del sistema y tecnologías seleccionadas para el desarrollo del MVP.

## Purpose

El propósito de esta documentación es definir de manera clara cómo será construido el sistema AGE, tanto a nivel visual como técnico, asegurando coherencia entre la experiencia del usuario, la arquitectura del sistema y las herramientas seleccionadas.

Esto permite al equipo de desarrollo tener una guía clara para la implementación del MVP.
